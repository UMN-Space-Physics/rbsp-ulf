Abstract


   CSPICE_SPKOPA opens an existing SPK file for subsequent write. 

   For important details concerning this module's function, please refer to
   the CSPICE routine spkopa_c.

I/O

   
   Given the name of an existing SPK file, 'spk_ker', the call:
   
      cspice_spkopa, spk_ker, handle
   
   opens the file for write operation, returning an integer reference
   'handle' attached to the SPK file. Use 'handle' as the file
   reference for subsequent file operations.
   
   Use this routine only when appending additional segments to
   an SPK file.
