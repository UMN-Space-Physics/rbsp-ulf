Abstract


   CSPICE_EKOPW opens an existing E-kernel file for writing.

   For important details concerning this module's function, please refer to
   the CSPICE routine ekopw_c.

I/O

   
   Given:
   
      fname    the scalar string holding the name of the EK file to 
               open.
   
   the call:
   
      cspice_ekopw, fname, handle
   
   returns
   
      handle   a scalar integer that refers to the EK file.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

      ;;
      ;; Set the name for the EK file.
      ;;   
      EK = 'ekfile.ek'
   
      ;;
      ;; Open the EK for write. Icy signals an error if the file does
      ;; not exist.
      ;;
      cspice_ekopw, EK, handle
   
         ... write data, but don't alter the value of handle ...
   
      ;;
      ;; SAFELY close the file. Always close an EK file opened 
      ;; for writing with cspice_ekcls.
      ;; 
      cspice_ekcls, handle
