Abstract


   CSPICE_DET returns a scalar double describing the determinant of a
   3x3 matrix.

   For important details concerning this module's function, please refer to
   the CSPICE routine det_c.

I/O


   Given:

      mat   an arbitrary 3x3 double precision matrix.

   the call:

      det = cspice_det( mat )

   returns:

      det   double precision scalar determinant of 'mat'.

Examples


   Native IDL code to calculate the same scalar result:

      det = determ( mat )

         or

      det = la_determ( mat )

   The IDL functions accepts an arbitrary size NxN matrix.
