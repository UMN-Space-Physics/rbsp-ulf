Abstract


   CSPICE_VREL returns the relative difference between two
   double precision 3-vectors.

   For important details concerning this module's function, please refer to
   the CSPICE routine vrel_c.

I/O


   Given:

      vec1,
      vec2   two double precision 3-vectors.

   the call:

      diff = cspice_vrel( vec1, vec2 )

   returns:

      diff   the scalar double precision relative difference of the
             two vectors.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.


      vec1 = [ 12.3d   , -4.32d ,  76.d  ]

      vec2 = [ 23.0423d, -11.99d, -0.10d ]

      diff = cspice_vrel( vec1, vec2 )

      print, diff

   IDL outputs:

      1.0016370
