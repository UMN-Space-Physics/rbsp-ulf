Abstract


   CSPICE_VSUBG returns a N vector double describing the difference
   of two N-vectors.

   For important details concerning this module's function, please refer to
   the CSPICE routine vsubg_c.

I/O


   Given:

      vec1,
      vec2    two arbitrary double precision N-vectors.

   the call:

      cspice_vsubg, vec1, vec2, diff

   returns:

      diff   the double precision N-vector subtraction of 'vec1' and 'vec2'.

Examples


   Native IDL code to calculate the same vector result:

      diff = vec1 - vec2

   The IDL '-' operator accepts arbitrary size N vectors.

   None.
