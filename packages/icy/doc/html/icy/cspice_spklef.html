Abstract


   CSPICE_SPKLEF loads an ephemeris file for use by SPK readers then 
   returns an integer file handle. 

   For important details concerning this module's function, please refer to
   the CSPICE routine spklef_c.

I/O

   
   Given:
   
      fname   a scalar string SPK file name
      
    the call:
   
      cspice_spklef, fname, handle
   
   returns:
   
      handle   a scalar integer handle assigned to the file upon loading.
               Almost every other SPK routine will subsequently use
               this number to refer to the file.
   
   Note: This routine, and all kernel file load routines requiring 
   explicit file handle management has been superseded by 
   cspice_furnsh. NAIF recommends users load all kernel files for 
   read-only data access with cspice_furnsh.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

   
      ;;
      ;; Load the spk file, return the integer value
      ;; for the handle.
      ;;
      
      cspice_spklef, SPK, spkhan
   
      ... perform calculations, do not alter value of handle ...
   
      ;;
      ;; Done. Unload the SPK file.
      ;;
      cspice_spkuef, spkhan
